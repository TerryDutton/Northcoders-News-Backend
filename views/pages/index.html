Northcoders News

API Endpoints

GET /api/topics

Get all the topics.

GET /api/topics/:topicID/articles

Get all articles of a certain topic.

POST /api/topics/:topicID/articles

Add a new article to a topic. This requires a JSON body with keys of title, body and created_by, constructed like so: {"title": "My Title Here", "body": "My article contents here", "created_by": "My userID here"}.

GET /api/articles

Get all articles.

GET /api/articles/:articleID

Get an individual article.

GET /api/articles/:articleID/comments

Get all comments for an individual article.

POST /api/articles/:articleID/comments

Post a comment to an article. This requires a JSON body with keys of body and created_by, constructed like so: { "body": "My comment here", "created_by": "My userID here" }.

PUT /api/articles/:articleID?vote=up

Increment or decrement an article's vote count. This route requires a vote query of either "up" or "down".

PUT /api/comments/:commentID?vote=up

Increment or decrement a comment's vote count. This route requires a vote query of either "up" or "down".

DELETE /api/comments/:commentID

Delete a comment.

GET /api/users/:username

Retrieve a JSON object of a user's profile data.
